RGBA Widget

Features How To Use Demo Download

Features

  • Modify element css property colors without reloading the page
  • Easily find the best color for a css property by moving sliders
  • Movable/draggable
  • A tool to be used during development

How To Use

Get the widget onto the screen by adding the js and css into the page:

	<link rel="stylesheet" href="rgba-widget.css" />
	<script src="rgba-widget.js"></script>

With the widget on the page:

  1. Enter a selector 
    	As you would with document.querySelectorAll
  2. Enter a css property 
    	For the elements matching the selector
  3. Adjust the 4 sliders 
    	To find the best rgba color for the property
